Installation
============
Using [Homeshick](https://github.com/andsens/homeshick):
```bash
git clone https://github.com/andsens/homeshick ~/.homesick/repos/homeshick
~/.homesick/repos/homeshick/bin/homeshick clone http://github.com/zackp30/dotfiles
```
